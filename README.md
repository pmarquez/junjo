# junjo
junjo: "sequence" in Japanese, is a microservice that returns the next code in, well, a sequence or series.

## Requirements
* JBoss Undertow 2.2.14.Final or newer
* MongoDB 4.0.9 or newer
* Spring Boot 2.6.2
* Java 17
